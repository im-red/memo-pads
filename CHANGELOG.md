## [v1.0.1] - 2026-04-30

### Improvements

- **Splash Screen** - Updated splash screen images across all device sizes and orientations for better visual quality. Changed splash screen configuration to manually hide for smoother app launch experience.

- **Repository Naming** - Renamed repository from `memo_pads` to `memo-pads` for consistency with modern naming conventions. Updated all references in documentation, settings, and about page.

- **Export File Naming** - Updated export file naming convention from `memo_pads_` to `memo-pads_` for consistency with the new repository name.

## [v1.0.0] - 2026-04-15

### New Features

- Initial release of Memo Pads application
- Create and manage multiple notebooks for organizing vocabulary
- Add, edit, and delete memo cards with front and back content
- Swipe navigation between memo cards
- Import and export functionality for data backup
- WeRead import support for importing highlights
- Trash bin for deleted items with restore functionality
- Settings page with app information and update checker
- About page with app details and GitHub repository link
